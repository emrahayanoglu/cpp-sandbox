# KLab/PlaygroundOSS

Required [AndroidNDK](../android-ndk).

``` sh
# rebuild engine
git clone https://github.com/KLab/PlaygroundOSS.git
cd PlaygroundOSS/Engine/porting/Android/GameEngine-android
NDK_PROJECT_PATH=`pwd` ./build.py --rebuild --project SampleProject                                                                                                                                    
```
